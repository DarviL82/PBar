PBar

Crea barras de progreso en la terminal con Python

Fácil de usar

PBar es realmente fácil de usar. Tan solo es necesario establecer el rango de progreso, y establecer los puntos en los que la barra progresará, nada más.

Personalizable

¿Quieres cambiar el color de la zona completada? ¿Mostrar el porcentaje como título de la barra? ¿O simplemente mostrar una barra vertical? Sin problemas, con PBar podrás personalizar casi todos los aspectos que puedas imaginar.

Completo

PBar cuenta con unas cuantas funciones y objetos que te ayudarán a la hora de gestionar tareas con la barra.

Características

Espera, ¿Seguro que es sencillo?

import pbar b = pbar.PBar() b.draw()
Aquí tienes el código que genera una barra de carga con los valores por defecto. También muestra la muestra en la terminal. Así que bueno... ¡Debería ser sencillo!

Vale vale, pero me interesa que sea algo útil.

Eso tiene sentido, a todo el mundo le gustan las cosas útiles, (al menos eso creo). Por eso, puedes mostrar el progreso de un conjunto de tareas de forma súper sencilla.
@pbar.taskWrapper(b) def my_task():     get_data()     process_stuff()     show_data()     useless_function()     another_function()

Me dan miedo los @decoradores, ¿Alguna otra opción?

def my_task():     get_data()     b.step()     process_stuff()     b.step()     show_data()     b.step()     useless_function()     b.step()     another_function()     b.step()

¡Pues claro! ¿A quién no le da miedo un decorador? Son verdaderamente terroríficos, y por eso, puedes seguir progresando tu barra de la forma 'clásica'.

Eso sí, me temo que tendrás que admitir que es mucho más feo.

¿Y si quiero cambiar el color cuando se complete?

Es tu día de suerte, por que puedes establecer condiciones para bastantes propiedades de la barra, y que cuando se cumplan, ¡se apliquen diferentes colores, caracteres, o texto!
conds = (     pbar.Cond("percentage >= 50", pbar.ColorSet.YELLOW, pbar.CharSet.BRICKS),     pbar.Cond("percentage == 100", pbar.ColorSet.GREEN),     pbar.Cond("text <- error", pbar.ColorSet.RED, formatset=pbar.FormatSet.DESCRIPTIVE) ) b = pbar.PBar(conditions=conds) try:     pbar.animate(b, range(50)) except:     b.text = "Error!"     b.draw()
